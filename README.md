# 🎬 Projeto Final - Catálogo IMDB

Repositório com os arquivos do projeto Catálogo IMDB, utilizando Java, estruturas de dados e conceitos de Programação Orientada a Objetos (POO).

## Enunciado do Projeto

Desenvolva um sistema que apresente um catálogo de filmes similar ao IMDB.

Neste sistema, cada filme deve possuir alguns atributos como o nome, data de lançamento, orçamento e descrição.

Além disso, cada filme também deve possuir um diretor e uma lista de atores. Adicione atributos para as pessoas. Implemente uma aplicação em que o usuário possa:

* Cadastrar filmes
* Cadastrar atores
* Cadastrar diretores
* Associar um filme aos seus respectivos atores e diretores;
* Pesquisar filme cadastrados pelo nome (desconsiderando letras maiúsculas e minúsculas).

Utilize os conceitos de Programação Orientada a Objetos como encapsulamento, herança, polimorfismo e classes abstratas.

## Funcionalidades

* Cadastro:Registro de Filmes, Diretores e Atores.
* Associações: Vínculo de um Diretor e múltiplos Atores a um Filme.
* Listagem: Visualização de todos os Filmes, Diretores e Atores cadastrados.
* Pesquisa: Busca de filmes por nome com exibição detalhada.
* Interface: Menu interativo e colorido via terminal.
